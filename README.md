# 🤖 QnA Chatbot with LangChain + Streamlit

A simple Question & Answer chatbot built using LangChain and Google's Gemini model with a Streamlit interface. The chatbot provides structured responses including answers, confidence levels, reasoning, and suggested follow-up questions.

## Features

- Ask natural language questions
- Structured AI responses using Pydantic schema
- Confidence score for answers
- Reasoning behind responses
- Suggested follow-up questions
- LangSmith tracing and observability support
- Simple Streamlit chat interface
- Batch question support

## Tech Stack

- Python
- LangChain
- Gemini 2.5 Flash Lite
- Streamlit
- LangSmith
- Pydantic

## Setup

Clone the repository:

```bash
git clone <your-repo-url>
cd <project-name>
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
GOOGLE_API_KEY=your_google_api_key
LANGSMITH_API_KEY=your_langsmith_api_key
LANGSMITH_TRACING=true
LANGSMITH_PROJECT=qna-chatbot
```

## Run the Application

Start the Streamlit app:

```bash
streamlit run app.py
```

Open the local URL generated by Streamlit in your browser.
